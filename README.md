## Usage

### Maven

```xml
<dependency>
    <groupId>io.github.suppierk.ddd</groupId>
    <artifactId>cqrs</artifactId>
    <version>1.319.0</version>
</dependency>
```

### Gradle

```groovy
implementation 'io.github.suppierk.ddd:cqrs:1.319.0'
```
